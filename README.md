# ETL Validation Framework

## Features
- Metadata-driven framework
- Excel & ETL validation
- Data comparison with tolerance
- Column-level mismatch detection
- HTML dashboard reporting
- Allure reporting
- Email automation

## Tech Stack
- Python
- Pandas
- Pytest
- Allure
- Matplotlib

## How to Run
1. Set environment variables:
   EMAIL_USER
   EMAIL_PASSWORD

2. Run:
   python main.py
